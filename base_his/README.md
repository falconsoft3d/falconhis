# FalconHIS V19
FalconHIS is a comprehensive Hospital Information System (HIS) designed to streamline healthcare management and improve patient care. It offers a wide range of features including patient registration, appointment scheduling, medical records management, billing, and reporting.

# Hospital Structure
The hospital structure in FalconHIS is organized hierarchically to facilitate efficient management of hospital resources
```
Hospital → Building → Floor → Room (Sala) → Bed (Cama)
Hospital → Edificio → Planta → Sala → Cama
```
